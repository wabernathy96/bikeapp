# Bike App

A data aggregation application built in order to display information about biking in Houston, Texas.

Technology Used: React, Redux, Node, Express, React-leaflet, Bikewise API

For More Information: https://docs.google.com/document/d/13JHg9Jtga13KDZeWVld3BsSGzL5PSlhgM5lWClpwiyk/edit?usp=sharing

## Getting Started

1) Clone repository: git clone git@github.com:wabernathy96/bikeapp.git

2) Install root dependencies: npm i

3) Install dev dependencies: npm i --dev

4) Run from root: npm run dev
